NOTAM Parser
============

This is a `NOTAM <https://de.wikipedia.org/wiki/NOTAM>`_ parser for Python 3.
The goal is to support the ICAO format.

Dependencies
------------

- Python 3
- PLY

Example
-------

::

    python3 -m notam.parser test_files/notam-A0513-14.txt
